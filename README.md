### How to run

+ Clone the repository and install dependencies:

   ```
   $ git clone git@github.com: Stint / https://github.com/Suspir0n/Stint.git  
   $ cd Stint  
   ```
   
+ Run the application:

    `$ ionic cordova run android`
    
+ Download the apk:

     `$ ionic cordova build android`
